# Mixed Zones Mode - True Interleaving

## Before (Sequential - All tasks from one zone at a time):
```
Kitchen:
  1. Clear clutter from counters
  2. Deep clean stovetop
  3. Clean microwave inside/out
  4. Wipe down fridge exterior
  ... (all 13 kitchen tasks)

Bathroom:
  1. Pick up items from floor
  2. Clean and organize vanity
  ... (all 13 bathroom tasks)

Bedroom:
  1. Clear clutter from all surfaces
  ... (all 15 bedroom tasks)
```

## After (Round-Robin Interleaving - ADHD-Friendly Variety):
```
1. Kitchen → Clear clutter from counters
2. Bathroom → Pick up items from floor
3. Bedroom → Clear clutter from all surfaces
4. Living Room → Deep vacuum couch cushions
5. Laundry Room → Sort dirty laundry

6. Kitchen → Deep clean stovetop
7. Bathroom → Clean and organize vanity
8. Bedroom → Change bed sheets
9. Living Room → Lint roll all furniture
10. Laundry Room → Check for stains

11. Kitchen → Clean microwave inside/out
12. Bathroom → Clean mirror
13. Bedroom → Organize closet
14. Living Room → Wash dog blankets
... and so on
```

## Benefits:
✅ **Prevents boredom** - You're never stuck in one room for too long  
✅ **ADHD-friendly** - Constant variety keeps focus and motivation high  
✅ **Natural breaks** - Moving between zones gives mini mental resets  
✅ **Progress feels faster** - Seeing multiple zones get cleaner simultaneously is more rewarding
